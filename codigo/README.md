# Código

Esta carpeta contiene los scripts utilizados para el análisis del proyecto.

Los códigos se organizan según las etapas principales del trabajo:

- Limpieza y preprocesamiento de datos.
- Análisis descriptivo de pacientes.
- Análisis geoespacial y accesibilidad.
- Transiciones diagnósticas mediante códigos CIE-10.
- Modelos de redes bayesianas.
